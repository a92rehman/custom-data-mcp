2 Install Claude Code

Run this in your terminal (PowerShell on Windows, Terminal on Mac/Linux).

$ npm install -g @anthropic-ai/claude-code

Then verify it works:

$ claude --version

First time? Run claude in your terminal — it will walk you through Anthropic authentication. Sign in with your Anthropic account.

3 Install the Data MCP Plugin

Two commands to add the marketplace and install the plugin.

# Add the Orenda-Project marketplace $ claude plugin marketplace add Orenda-Project/taleemabad-data-mcp # Install the plugin $ claude plugin install taleemabad-data@Orenda-Project

Git access required. The repository is private. If you get a git error, ask IT to add your GitHub account to the Orenda-Project organization.

4 Run Setup

One-time setup to save your email and sync governance rules.

Open Claude Code

Open your terminal and type claude to start Claude Code. You can be in any project directory.

Run the setup command

Type /taleemabad-setup and press Enter. It will ask for your work email.

> /taleemabad-setup

Restart Claude Code

Close and reopen Claude Code, or press Ctrl+R to reload.

Verify connection

Type /mcp — you should see taleemabad-data · connected

Then try: "How many active PRIMARY teachers are in ICT/Islamabad?"

6 Updating

Rules auto-sync every session. Manual updates are rarely needed.

WhatHowWhen
Rules & Agents Automatic Every Claude Code session start
Plugin version claude plugin update taleemabad-data@Orenda-Project When told a new version is out, or if things break
Check version Ask: "What MCP version am I running?" Any time

Update stuck on Windows? Close ALL Claude Code windows first, then retry. The plugin directory gets locked by running processes. If still stuck, uninstall and reinstall (see Troubleshooting).
